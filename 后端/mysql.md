## 常用命令

| 命令                       | 作用                            |
| -------------------------- | ------------------------------- |
| mysql -u root p            | 连接服务器                      |
| show databases;            | 展示所有数据库                  |
| create database dbname;    | 新建数据库                      |
| show tables;               | 展示该库下的所有表              |
| create table test(id int); | 创建test表，有个int类型的id属性 |
| drop table test;           | 删除test表                      |
| exit                       | 退出mysql                       |
| select * from tablename    | 查询表信息                      |
|                            |                                 |
