# webstrom 

- 按住 alt ，点击鼠标右键，可以出现多个光标

- 按住`Ctrl + Alt + T`，可以选择选项，将这段字符环绕

- eslint: 按住  `Ctrl + Shift + Alt + P` , Reformat  with Prittier  (用pritter重新格式化)

- 对选中字符串大小写转化：`  Ctrl + Shift + U `   。如果有大写字母则转化小写，如何全是小写，则全转化位大写。

- 深度折叠选中代码块

  - ``` 
    Ctrl + Shift +   “+”    // 折叠
    ```

  - ```
    Ctrl + Shift +   “-”    //展开
    ```
