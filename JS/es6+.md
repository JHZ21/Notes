## 结构赋值

> 可以将属性/值从对象/数组中取出,赋值给其他变量。
>
> ```js
> [a, b] = [10, 20]
> // 左边接受方，必须写明属性
> var x = [1, 2, 3, 4, 5];
> var [y, z] = x;
> ```



## export  and import

```javascript
// 导出不用 as xxx
export * from "./permission"

// 导入 需要 as xxx
import * as directives from "@/directives"
```
